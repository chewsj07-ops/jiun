Privacy Policy (隐私政策)

Last Updated (最后更新): 2026-03-11

ZenAssistant (禅心助手) ("we", "us", "our") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, store, and protect your information when you use the ZenAssistant application and related services.

ZenAssistant（禅心助手）（以下简称“我们”）致力于保护您的隐私。本隐私政策说明在您使用 ZenAssistant 应用程序及相关服务时，我们如何收集、使用、存储和保护您的信息。

ZenAssistant is an application platform for meditation, awareness, reflection, and spiritual growth. We understand that the content users input can be highly personal, so protecting this data is our core principle.

ZenAssistant 是一个用于 冥想、觉察、反思与身心成长 的应用平台。我们理解用户输入的内容可能具有高度个人性，因此保护这些数据是我们的核心原则。

1. Information We Collect (我们收集的信息)

Account Information (账户信息)

When you register an account, we may collect (当您注册账户时，我们可能收集):

  • Email address (电子邮件地址)
  • Username / display name (用户名/昵称)
  • Encrypted password (加密的密码)

Passwords are never stored in plain text. (密码不会以明文存储。)

User Content (用户内容)

Users can record the following in the app (用户可以在应用中记录):

  • Thought observations (念头觉察)
  • Meditation records (禅修记录)
  • Gratitude or good deeds (感恩或善行)
  • Dedications or vows (回向或发愿)
  • Practice progress (修行进度)

This content belongs to the user's personal reflection data and is only used for app functionality. (这些内容属于用户个人反思数据，仅用于应用功能。)

Technical Information (技术信息)

The system may collect (系统可能收集):

  • Device type (设备类型)
  • IP address (IP地址)
  • App usage logs (应用使用日志)
  • Error reports (错误报告)

2. How We Use Information (我们如何使用信息)

  • Creating user accounts (创建用户账户)
  • Saving practice records (保存修行记录)
  • Cross-device synchronization (跨设备同步)
  • AI reflection assistance (AI反思辅助)
  • Improving app experience (改进应用体验)
  • Customer support (客户支持)
  • Security monitoring (安全监控)

ZenAssistant does not sell user data. (ZenAssistant 不会出售用户数据。)

3. AI Processing (AI 处理)

Some features use AI technology to generate reflections or suggestions. User input content may be sent to secure AI services for processing to generate responses. User data will not be used to train public AI models unless explicit authorization is obtained.

4. Data Ownership (数据所有权)

All practice records belong to the user. Users have the right to view, modify, export, and delete data, as well as delete their account.

5. Data Security (数据安全)

We use industry-standard security measures including HTTPS encryption, secure authentication, and encrypted password storage.
